# 🏒 Бот для хоккейной барахолки

Бот перехватывает объявления в группе, просит подписаться на каналы,
и только после проверки публикует объявление.

---

## ⚙️ Настройка (5 шагов)

### 1. Создай бота
- Напиши @BotFather → /newbot → получи токен
- Вставь токен в `bot.py` → строка `BOT_TOKEN = "..."`

### 2. Узнай ID группы
- Добавь @userinfobot в группу → напиши /start
- Скопируй ID (отрицательное число, например -1001234567890)
- Вставь в `bot.py` → строка `GROUP_ID = ...`

### 3. Узнай свой Telegram ID
- Напиши @userinfobot → /start в личке
- Вставь в `bot.py` → строка `ADMIN_IDS = [твой_id]`

### 4. Добавь бота в группу и каналы
- Добавь бота **администратором** в свою барахолку
  (нужны права: удаление сообщений)
- Добавь бота **администратором** в каждый хоккейный канал
  (нужны права: просмотр участников)

### 5. Замени каналы в channels.json
Отредактируй файл `channels.json` — вставь свои каналы:
```json
[
  {
    "username": "@твой_канал",
    "name": "Название канала",
    "url": "https://t.me/твой_канал"
  }
]
```

---

## 🚀 Запуск

```bash
pip install -r requirements.txt
python bot.py
```

---

## 📋 Команды администратора

| Команда | Действие |
|---------|----------|
| `/listchannels` | Показать все каналы |
| `/addchannel @username Название https://t.me/username` | Добавить канал |
| `/removechannel @username` | Удалить канал |
| `/stats` | Сколько объявлений ждут публикации |

### Пример добавления канала:
```
/addchannel @hockey_russia Хоккей России https://t.me/hockey_russia
```

## ❓ Частые проблемы

**Бот не удаляет сообщения**
→ Дай боту право "Удаление сообщений" в настройках группы

**"Не смог написать в личку"**
→ Пользователь ещё не начинал диалог с ботом.
  Бот попросит его написать боту — это нормально, один раз.

**Бот не проверяет подписку**
→ Убедись что бот добавлен администратором в канал
